# Terraform + Docker (Remote Context) + MySQL 8

Проект демонстрирует, как с помощью Terraform управлять Docker‑контейнерами на удалённой виртуальной машине (Yandex Cloud) через SSH‑remote context.  
В качестве примера разворачивается контейнер **MySQL 8** с автоматически сгенерированными паролями.

---

## Возможности проекта

- Подключение Terraform к удалённому Docker через SSH  
- Генерация паролей с помощью `random_password`  
- Передача ENV‑переменных в контейнер  
- Проброс порта `127.0.0.1:3306`  
- Полностью детерминированный и воспроизводимый запуск MySQL
